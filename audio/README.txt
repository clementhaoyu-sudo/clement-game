Musique Beast Game (beast-game.mp3)
===================================

Pour avoir la musique en jeu :
1. Télécharge une musique libre (style jeu / Beast) en MP3.
2. Enregistre le fichier sous le nom : beast-game.mp3
3. Place-le dans ce dossier (audio/).

Exemple de musique libre « jeu » :
- Pixabay (gratuit, sans compte) : https://pixabay.com/music/search/game/
- Choisis une piste, clique sur « Download », renomme le fichier en beast-game.mp3 et mets-le ici.

Si beast-game.mp3 est absent, le jeu tentera une piste de secours en ligne (selon le code).
